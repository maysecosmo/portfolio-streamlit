# 💻 Portfólio Tech & Editorial | Mayse Cosmo

Este projeto é uma aplicação web desenvolvida com **Python** e **Streamlit**, criada para unir minha sólida trajetória de 8 anos no mercado editorial com minha transição para o desenvolvimento de software (ADS).

---

## ✨ O Projeto
O objetivo desta aplicação é oferecer uma interface funcional onde clientes e recrutadores podem conhecer meu trabalho, conferir tabelas de serviços e simular orçamentos de forma automatizada.

### 🛠️ Funcionalidades
* **Sobre Mim:** Resumo da minha jornada de transição de carreira (Letras para ADS).
* **Serviços e Prazos:** Tabela dinâmica de serviços editoriais.
* **Orçamento Inteligente:** Simulador que utiliza lógica de programação para calcular estimativas de projetos com base no volume de páginas ou laudas.
* **Interface Personalizada:** Design clean utilizando tons de lilás e neutros, configurados via TOML.

## 🚀 Tecnologias Utilizadas
* **Linguagem:** Python
* **Interface:** Streamlit
* **Dados:** Pandas
* **Deploy:** Streamlit Community Cloud

2. Clone este repositório.
3. Instale as dependências:

4. Execute a aplicação:
